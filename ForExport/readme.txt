Данный набор программ представляет собой комплекс, включающий в себя приложение потокового копирования файлов и программу установки, работающую с сжатыми Rar архивами.

Текущие исходники содержат следующие примеры:

- работа с потоками (создание, запуск, останов);
- работа с файлами (потоковое копирование, вычисление контрольных сумм, отображение прогресса работы а также докачка файлов);
- работа с системным реестром (контекстное меню);
- работа с буфером обмена;
- работа с архивами;

Сам архив с файлами именуется "Data.dat", его использует программа - инсталлятор.
Для работы необходимо установить программу, затем загрузить в буфер обмена файлы (выделив и нажав Ctrl+C). Далее копирование осуществляется через контекстное меню папки или диска, куда необходимо скопировать выделенные файлы.

Почти аналог killCopy.
